# Country Capital Finder

Модуль для консольных приложений на C#, возвращающий название страны и её столицу по буквенному коду.

<img width="408" height="117" alt="Снимок экрана 2026-03-13 203156" src="https://github.com/user-attachments/assets/e8fe04ab-430b-4b26-a3a7-3d2305ec37d5" />

## Функциональность

- Ввод буквенного кода страны
- Вывод названия страны и столицы:
  б → Беларусь, Минск
  р → Россия, Москва
  л → Литва, Вильнюс
- Обработка некорректного ввода

## Пример работы

Введите название страны: б, р, л
р
Россия, Москва

Введите название страны: б, р, л
у
Некорректная буква страны
